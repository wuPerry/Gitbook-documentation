# 案例

### 本地（window）：

##### 步骤一：安装nodejs

安装教程参考以下链接:

[http://www.runoob.com/nodejs/nodejs-install-setup.html](http://www.runoob.com/nodejs/nodejs-install-setup.html)

##### 步骤二：安装gitbook

打开运行,输入以下命令：

```
npm install -g gitbook-cli
```

![](/assets/import15.png)

检查安装：

在运行窗口输入以下命令检查gitbook是否安装成功

```
gitbook -V
```
